# COS-202 Python Programming Assignments

This repository contains my COS-202 (Python Programming) assignments, showcasing Python applications developed to demonstrate fundamental programming concepts and problem-solving skills.

The repository includes two projects:

1. **Mathematical Calculator (MC)**: A simple graphical calculator built with Python's Tkinter library that performs basic arithmetic operations, including addition, subtraction, multiplication, division, integer division, exponentiation, modulus, as well as Clear and OFF functions.

2. **Personal Pocket CGPA Calculator (PPC)**: A console-based application that calculates a student's CGPA on a 5.0 grading scale. The program uses Python's selection control statements ("if", "elif", and "else") to determine grades, compute grade points, calculate the cumulative CGPA, and display the appropriate degree classification.

These assignments were developed as part of the requirements for the COS-202 Python Programming course and demonstrate the practical application of Python fundamentals, including user input, arithmetic operations, control statements, loops, functions, and graphical user interface (GUI) development.
